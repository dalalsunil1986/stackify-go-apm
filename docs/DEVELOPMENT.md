# Go APM Development Guide 

## Development Setup

1. Install Go 1.15

2. Clone Repository
    http: `git clone https://<user>@bitbucket.org/stackify/stackify-go-apm.git`
    ssh:  `git clone git@bitbucket.org:stackify/stackify-go-apm.git`

3. Install Dependencies
    `$ go mod tidy`
